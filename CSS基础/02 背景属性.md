# 1.常见背景属性
background-color:  背景颜色

background-image: url  将图像设置为背景

**background-repeat: 设置背景图片是否重复及如何重复，默认平铺满**
  
  no repeat 不要平铺
  
  repeat-x 横向平铺
  
  repeat-y 纵向平铺

background-position:center top 设置背景图片在当前容器中的位置

background-attachment:scroll  设置背景图片是否跟着滚动条一起移动 fixed

background:将上面的多个属性写在一个声明中

**background-origin** 

**background-clip 背景裁剪**

**background-size 调整尺寸**

多重背景

# 2.background-color 背景颜色的表示方法

## 2.1 单词

  英语单词表示简单颜色
  
  background-color: red;
  
## 2.2 RGB表示法

  三原色
  
  ```
  background-color: rgb(255,0,0);
  ```
  
  黑色
  
  ```
  background-color: rgb(255,255,0)
  ```
  
## 2.3 RGBA表示法--红色，绿色，蓝色，透明度
  
    前三个取值为(0,255),透明度取值为(0,1)

## 2.4 十六进制表示法

 #ff0000就是RGB(255,0,0)
 
 几种常见的颜色:
 
 #000000 黑色
 
 #ffffff 白色
 
 #ff0000 红色
 
 #222222 深灰
 
 #333333 灰色
 
 #cccccc 浅灰
 
## 2.5 HSLA表示法

色调：0~360；0或360表示红色，120表示绿色，240表示蓝色

饱和度： 0%~100%，值越大，越鲜艳

亮度:0%~100%，亮度最大时为白色，最小时为黑色

透明度：0~1

# 7.background-size属性 背景尺寸

  ```
  background-size:500px 500px;
  background-size:50% 50%;
  background-size:100% auto;
  
  background-size:cover; /*图片始终填充满容器，如果有超出部分会被隐藏*/
  
  background-size:contain' /*将图片完整显示在容器中，保证长宽比不变，可以有部分区域为空白*/
  ```
  
# 8.background-origin属性 

  控制背景从什么地方开始显示
  
  ```
  /*从内边距开始显示背景图*/
  background-origin: padding-box;
  /*从边框开始显示*/
  background-origin: border-box;
  /*从content-box内容区域开始显示*/
  background-origin: content-box;
  ```
# 8.background-clip属性
  
  设置元素背景是否延伸到边框下面
  
  background clip: content-box;  超出部分将被裁剪掉
  
  boredr-box  超出boredr-box部分将裁剪掉
  
  padding-box

# 10.background-image属性渐变
  
  创建一个只有元素的部分区域可以显示的剪切区域，区域内的部分显示，区域外的隐藏
  举例：鼠标悬停时，方法裁剪区域
  ```
  <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <!--
        1.常见背景属性
          background-color:  背景颜色
          background-image: url  将图像设置为背景
          background-repeat: 设置背景图片是否重复及如何重复，默认平铺满
    -->
    <!--
        background-repeat属性
    -->
    <!--
        渐变

    -->
   <!--
      clip-path:裁剪出元素的部分区域做展示
    鼠标悬停时，放大裁剪区域
     -->
    <style>
        .img{
            width: 200px;
            height: 200px;
            border: 1px solid red;
            background: url(http://img.smyhvae.com/20191006_1410.png) no-repeat;
            background-size: cover;
        }
        .div1{
            clip-path: polygon(50% 0%, 100% 50%, 50% 100%,0% 50%);
            margin-bottom: 20px;
        }
        .div2{
            clip-path: polygon(50% 0%, 100% 50%, 50% 100%,10% 10%);
        }
    </style>
    
</head>
<body>
    <div class="img div1"></div>
    <div class="img div2"></div>
</body>
</html>
  ```
